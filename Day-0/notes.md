## Befehle + Erklärung + Beispiele (alles zusammen)

```bash
# whoami
# Bedeutung:
# Zeigt den aktuell angemeldeten Benutzer

# Beispiel:
whoami

# Ergebnis:
mehdi


# pwd
# Bedeutung:
# Zeigt den aktuellen Pfad

# Beispiel:
pwd

# Beispiel-Ausgabe:
/home/mehdi


# ls
# Bedeutung:
# Zeigt den Inhalt eines Ordners

# Beispiel:
ls

# Ergebnis:
Bilder Dokumente Downloads


# cd
# Bedeutung:
# Wechselt in ein Verzeichnis

# Beispiel:
cd mehdi


# cd ..
# Bedeutung:
# Geht ein Verzeichnis zurück

# Beispiel:
cd ..


# mkdir
# Bedeutung:
# Erstellt einen neuen Ordner

# Beispiel:
mkdir CyberJourney


# touch
# Bedeutung:
# Erstellt eine Datei

# Beispiel:
touch notes.txt


# echo
# Bedeutung:
# Gibt Text aus oder schreibt Text

# Beispiel:
echo "Day 0 started"


# >
# Bedeutung:
# Überschreibt Dateiinhalt

# Beispiel:
echo "Day 0 started" > notes.txt


# >>
# Bedeutung:
# Fügt Text hinzu ohne zu löschen

# Beispiel:
echo "Learning Linux" >> notes.txt


# cat
# Bedeutung:
# Zeigt Inhalt einer Datei
cat notes.txt

---------------------------
